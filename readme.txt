== Description ==

DBpediaの説明文をブログ記事内に簡単に挿入できるプラグインです。
専門用語の多い記事などを作成する際にオススメです。

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= データソースは？ =

DBpediaからSPARQLで取得しています。
http://ja.dbpedia.org/

= 意図した説明文が出ないんだけど？ =

完全一致のSPARQL検索をしているので表記ゆれ等には対応していません。
